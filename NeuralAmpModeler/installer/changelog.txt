04/07/2026 - VoLum: Windows installer copies bundled Ampete rigs to {app}\rigs\Ampete One and sets HKLM\Software\VoLum\NeuralAmpModeler\AmpeteRigsPath for VST3. Ampete model menu for VST3 (IR browser unchanged). Defines VOLUM_AMPETE_PRODUCT in config.h.
04/08/2026 - VoLum: New Variant F UI layout (900x600). Sidebar amp browser, speaker mode buttons, hero image area, knob groups [Channel|Input,Gate|Bass,Mid,Treble|Output], I/O meters, toggles. Channel selector is a discrete stepper (not a knob). Knob value labels update live from params. Dark procedural background. VoLumControls.h with custom controls. Original NAM layout preserved under #else.
04/08/2026 - VoLum: Callback-based rig selection. Channel stepper cycles only channels for the selected speaker mode (dynamic discovery via DiscoverChannels). Speaker buttons trigger reload. Sidebar amp clicks switch amp folder. Background model loading prevents UI freeze. Footer shows actual loaded .nam filename. Meter spacing increased. Font sizes bumped. _UpdateControlsFromModel guarded for VoLum mode. Plan and README updated with rig file patterns and architecture decisions.
04/08/2026 - v0.7.15 VoLum: Per-amp persistent settings. All knob, toggle, speaker, and channel settings are remembered per amp. Switching amps saves current settings and restores the target amp's settings. Settings persist to rigs/volum-settings.json across sessions. VST3 SerializeState/UnserializeState extended for per-amp settings array.
